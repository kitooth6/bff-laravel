**環境構築完全ガイド**

- Dockerfileの解説

- トラブルシューティング完全版

## 🎯 ドキュメントの使い方

### 初回環境構築時
1. [SETUP_GUIDE.md](SETUP_GUIDE.md) の手順に従って環境構築
2. トラブルが発生した場合は、同ドキュメントのトラブルシューティング参照

### 日常開発時
- メインの [README.md](../README.md) でクイックリファレンス
- 詳細が必要な場合は該当するガイドを参照

### チーム開発時
- 新メンバーには [SETUP_GUIDE.md](SETUP_GUIDE.md) を共有
- プロジェクト概要は [README.md](../README.md) を参照
